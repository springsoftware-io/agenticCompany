# Documentation

> **AI agents will generate comprehensive documentation here based on `PROJECT_BRIEF.md`**

## What Gets Generated

- **`knowledge_base/`** - Business context, requirements, domain knowledge
- **`architecture/`** - System design, ADRs, diagrams, patterns
- **`docs/`** - API specs, user guides, technical documentation

## How It Works

1. AI reads your `PROJECT_BRIEF.md`
2. AI extracts business context and requirements
3. AI documents technical decisions
4. AI creates comprehensive documentation
